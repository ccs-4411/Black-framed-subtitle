黑框影片與雙語字幕合併工具 (Video Black-Frame & Bilingual Subtitle Merger)
本專案是一款基於 Gradio 與 FFmpeg 開發的輕量級影音處理工具。專門解決雙語字幕（如中英字幕）在一般影片畫面中容易遮擋畫面、或因背景雜亂導致辨識度不佳的問題。

透過本工具，你可以一鍵為影片底部加上黑邊框，並將 SRT 雙語字幕完美渲染在黑框中，提升觀影與學習體驗！

✨ 核心特色
自動化 UI 連動：切換解析度（480p / 720p / 1080p）時，系統會自動調節最佳黑框高度、字幕字體大小與垂直位移，省去手動計算的煩惱。

高度自訂字幕：支援中英文分開設定字型、粗體、斜體、字體大小與顏色（內建色彩選擇器）。

動態 10 秒預覽：正式轉檔前，可自由指定時間軸（分鐘），快速生成 10 秒影片預覽，確保樣式滿意再切入全片轉檔。

極速渲染：後台底層將 SRT 轉換為高相容性的 ASS 字幕格式，並啟用 FFmpeg ultrafast 預設集與多執行緒優化。

🛠️ 環境需求與安裝
1. 安裝系統依賴 (FFmpeg)
本工具核心依賴 ffmpeg。請確保系統已安裝並將其加入環境變數：

Ubuntu/Debian: sudo apt update && sudo apt install ffmpeg

Mac (Homebrew): brew install ffmpeg

Windows: 請下載 FFmpeg 執行檔並配置到環境變數 Path 中（程式內亦包含 /usr/bin 等常見路徑的自動偵測修復）。

2. 安裝 Python 套件
請在終端機執行以下指令安裝所需套件：

Bash
pip install gradio
🚀 快速啟動
將你的原始碼儲存為 app.py，並在終端機執行：

Bash
python app.py
啟動後，瀏覽器會自動打開 http://127.0.0.1:7860 進入操作介面。

📖 介面功能與操作指南
整個工具的操作流程非常直覺，主要分為三個區塊：

📂 步驟 1：導入檔案與進階設定
選擇原始影片 (MP4)：上傳你要處理的影片。

選擇字幕檔案 (SRT)：上傳包含雙語字幕的 SRT 檔（第一行為中文，第二行為英文）。

📐 解析度與黑框進階設定（點擊展開）：

選擇解析度後，下方的黑框高度與垂直位移會自動調整至最佳推薦值，亦可手動微調滑桿。

🎨 設定中外文字幕样式（點擊展開）：

可獨立針對【中文】與【英文】設定字型（如微軟正黑體、Arial）、粗斜體、字體大小 (px) 以及顏色。

💡 小提示：設定解析度為開合式選單，設定完後黑框高度與雙語字幕字體大小皆會自動適配，大致上無需人工干預。

⚙️ 步驟 2：選擇工作流程
預覽起始時間（分鐘）：輸入你想查看的影片時間點（例如第 5 分鐘）。

🔄 生成 10 秒預覽：點擊後系統會快速截取該片段並渲染黑框與字幕，供你檢查效果。

🚀 執行全片轉檔：預覽滿意後，點擊此按鈕進行全影片的完整合併。

📺 步驟 3：查看成果與下載
右側的雙語字幕加黑邊預覽播放器會直接顯示渲染後的動態成果。

滿意成果後，可直接點擊播放器右上角的 ⬇️ 下載按鈕 儲存成品。

下方的 📝 系統日誌 (Console Log) 會即時回報 FFmpeg 的執行指令與處理進度。

⚙️ 程式碼核心架構說明
update_ui_components(resolution): 負責處理 UI 的動態連動。當使用者切換 1080p 或 720p 時，利用 gr.update() 即時變更滑桿的最大值與字體預設 px。

srt_to_ass(...): 將上傳的雙語 SRT 轉換為 ASS 格式。利用樣式表 ([V4+ Styles]) 分離 Default (中文) 與 SubTitle (英文) 的圖層與垂直邊距 (MarginV)，達成完美的雙語排版。

process_video_task(...): 核心轉檔封裝。利用 subprocess.run 呼叫 FFmpeg，並動態串接 scale (縮放)、pad (加黑邊) 以及 subtitles (掛載字幕) 的濾鏡鏈 (vf_chain)。
